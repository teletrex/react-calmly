# GeneralFlag

You can wrap content with `GeneralFlag` in order to display flag over any JSX.

## Example

<Canvas>
    <Story id="atoms-badge-general--usage" />
</Canvas>

## Component API

| Parameter name     |             Type             |  Default value  | Required  | Description          |
| -------------------| :--------------------------: | :-------------: | :-------: | :------------------: |
| children           | *OneOf(string, node)*        | `null`          | No        | `GeneralFlag` label  |
| color              | *string*                     | `'#1bf7a8'`     | No        | Color of the flag    |
