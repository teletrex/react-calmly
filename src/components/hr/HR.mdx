# Horizontal Line

## Default

By default `HR` does not need any props. If you want to apply some additional styling pass it by `className` props.

<Preview>
  <Story id="atoms-horizontalline--default" />
</Preview>

## Component API

| Parameter name |   Type   | Default value | Required | Description                                |
| -------------- | :------: | :-----------: | :------: | ------------------------------------------ |
| className      | _string_ |     `''`      |    No    | Optional `className` to be applied to `HR` |
